Wikidata Card Game Generator

Generate your own card game, suitable for playing Top Trumps, using Wikidata!

Here's how it works: Think of a topic, and type its singular form into the search box over at Wikidata. Then, look for the entity ID near the top, which starts with Q. For example, the ID of chemical element is Q11344. Add this ID to the URL, like this.

Here are some other topics you can try: capital, metro system, planet, hackerspace, computer model, card game.

Made for the Wikidata-Workshop 2019 in Ulm by bleeptrack and blinry. Source code is on GitHub, feel free to suggest ideas or report bugs!

If you like the generated card game, you can print it!
